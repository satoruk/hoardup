# Hoardup

## Overview

**hoardup** is a backup shell script.

* Using rsync

## Install

    curl -o hoardup https://raw.github.com/satoruk/hoardup/master/bin/hoardup
    chmod 755 hoardup

## Update

    $ hoardup self-update

## Quick start

    $ hoardup init backup
    $ cd backup
    $ hoardup backup
